معرض الأعمال

في هذه الصفحة يمكنك الاطلاع على أعمالي السابقة:

  • مشروع 1: تصميم واجهة المستخدم

    قمت بتصميم واجهة مستخدم لتطبيق موبايل يهدف إلى تسهيل عملية الطلبات عبر الإنترنت. تم استخدام HTML وCSS وJavaScript.

  • مشروع 2: تحليل بيانات التجارة الإلكترونية

    استخدمت Python وSQL لتحليل بيانات مبيعات عبر الإنترنت وتقديم رؤى استراتيجية حول سلوك العملاء.

  • مشروع 3: تطبيق الويب التفاعلي

    بناء تطبيق ويب تفاعلي باستخدام React وNode.js لعرض البيانات بشكل ديناميكي وتفاعل مع المستخدم بشكل سريع وفعال.
